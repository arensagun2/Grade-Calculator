# Grading Calculator

Created with React + Vite (JSX)

## Goal/Purpose

This calculator app will calculate your final grade for your class, with the scores and weights you provided.

## Installation

1. Clone Repo
2. CD to the Cloned Repo
3. ```npm install```

### To see a preview of the Cloned Repo

```npm run dev```
